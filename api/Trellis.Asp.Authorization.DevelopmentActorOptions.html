DefaultActorId

The unique identifier for the default fallback actor. Used when no X-Test-Actor header is present in the request. Defaults to "development".

public string DefaultActorId { get; set; }

DefaultPermissions

The permissions granted to the default fallback actor. Defaults to an empty set — override to grant permissions when the header is absent.

public IReadOnlySet<string> DefaultPermissions { get; set; }

Property Value

IReadOnlySet<string>

Examples

services.AddDevelopmentActorProvider(options =>
{
    options.DefaultPermissions = new HashSet<string>
    {
        "orders:create", "orders:read", "orders:read-all"
    };
});

ThrowOnMalformedHeader

When true, a malformed X-Test-Actor header throws InvalidOperationException instead of falling back to the default actor. Defaults to false.

public bool ThrowOnMalformedHeader { get; set; }
