Remarks

Security: This provider throws InvalidOperationException unconditionally in non-Development environments, preventing accidental deployment of the test bypass.

When no header is present, returns a configurable default actor (see DevelopmentActorOptions).

The header JSON schema matches the format produced by WebApplicationFactoryExtensions.CreateClientWithActor in Trellis.Testing: 

{
  "Id": "user-1",
  "Permissions": ["orders:create", "orders:read"],
  "ForbiddenPermissions": [],
  "Attributes": { "tid": "tenant-1" }
}

Register via AddDevelopmentActorProvider(IServiceCollection, Action<DevelopmentActorOptions>?).
