AddCachingActorProvider<T>(IServiceCollection)

Registers a caching decorator around the specified IActorProvider implementation. The inner provider is resolved per-scope and wrapped with CachingActorProvider so that multiple calls within the same request return the same actor instance.

public static IServiceCollection AddCachingActorProvider<T>(this IServiceCollection services) where T : class, IActorProvider

Parameters

services IServiceCollection

The service collection.

Returns

IServiceCollection

The service collection for chaining.

Type Parameters

T

The concrete IActorProvider implementation to cache.

Examples

// Wrap a DB-backed provider that resolves permissions from a database
services.AddCachingActorProvider<DatabaseActorProvider>();

AddClaimsActorProvider(IServiceCollection, Action<ClaimsActorOptions>?)

Registers ClaimsActorProvider as the scoped IActorProvider with configurable claim mapping for any OIDC/JWT identity provider.

public static IServiceCollection AddClaimsActorProvider(this IServiceCollection services, Action<ClaimsActorOptions>? configure = null)

Parameters

services IServiceCollection

The service collection.

configure Action<ClaimsActorOptions>

Optional delegate to customize ClaimsActorOptions. Override ActorIdClaim and PermissionsClaim to match your token format.

Returns

IServiceCollection

The service collection for chaining.

Examples

// Auth0
builder.Services.AddClaimsActorProvider(opts =>
{
    opts.ActorIdClaim = "sub";
    opts.PermissionsClaim = "permissions";
});

// Keycloak
builder.Services.AddClaimsActorProvider(opts =>
{
    opts.ActorIdClaim = "sub";
    opts.PermissionsClaim = "realm_access.roles";
});

AddDevelopmentActorProvider(IServiceCollection, Action<DevelopmentActorOptions>?)

Registers DevelopmentActorProvider as the scoped IActorProvider for development and testing environments. Reads actor identity from the X-Test-Actor HTTP header and falls back to a configurable default actor.

public static IServiceCollection AddDevelopmentActorProvider(this IServiceCollection services, Action<DevelopmentActorOptions>? configure = null)

Parameters

services IServiceCollection

The service collection.

configure Action<DevelopmentActorOptions>

Optional delegate to customize DevelopmentActorOptions. Set DefaultPermissions to grant permissions when no header is present, or ThrowOnMalformedHeader to reject invalid headers instead of falling back.

Returns

IServiceCollection

The service collection for chaining.

Examples

if (builder.Environment.IsDevelopment())
{
    builder.Services.AddDevelopmentActorProvider(options =>
    {
        options.DefaultPermissions = new HashSet<string>
        {
            "orders:create", "orders:read"
        };
    });
}
else
{
    builder.Services.AddEntraActorProvider();
}

Remarks

Security: The provider throws InvalidOperationException if the X-Test-Actor header is present in a Production environment. Use AddEntraActorProvider(IServiceCollection, Action<EntraActorOptions>?) for production deployments.

AddEntraActorProvider(IServiceCollection, Action<EntraActorOptions>?)

Registers EntraActorProvider as the scoped IActorProvider and configures EntraActorOptions with default Entra v2.0 claim mappings.

public static IServiceCollection AddEntraActorProvider(this IServiceCollection services, Action<EntraActorOptions>? configure = null)

Parameters

services IServiceCollection

The service collection.

configure Action<EntraActorOptions>

Optional delegate to customize EntraActorOptions. Override MapPermissions to flatten roles into granular permissions, MapForbiddenPermissions to populate deny lists, or MapAttributes to add custom ABAC attributes.

Returns

IServiceCollection

The service collection for chaining.

Examples

builder.Services.AddEntraActorProvider(options =>
{
    options.MapPermissions = claims => claims
        .Where(c => c.Type == "roles")
        .SelectMany(role => RolePermissionMap[role.Value])
        .ToHashSet();
});
