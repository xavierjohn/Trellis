Class HttpResponseExtensions

Namespace
Trellis.Asp
Assembly
Trellis.Asp.dll

The single Trellis verb for converting Result / Result<TValue> / WriteOutcome<T> / Page<T> values to ASP.NET Core HTTP responses. Works in both Minimal API endpoints and MVC controllers (.NET 7+ executes IResult natively in MVC). For typed ActionResult<T> signatures, chain AsActionResult<T>(IResult).

Remarks

Configure protocol semantics via the optional fluent HttpResponseOptionsBuilder<TDomain>: WithETag, WithLastModified, Vary, CreatedAtRoute, EvaluatePreconditions, HonorPrefer, etc. All selectors run against the domain value; the response body may be projected separately via the body overload.
