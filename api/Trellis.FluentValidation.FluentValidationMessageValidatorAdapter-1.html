Remarks

Registered as an open-generic IMessageValidator<TMessage> by AddTrellisFluentValidation(IServiceCollection). When no FluentValidation.IValidator<T> is registered for a given TMessage, the injected sequence is empty and the adapter returns success without allocating violations.

FluentValidation property names that include member chains (Address.PostCode) or indexers (Items[0].Sku) are translated to RFC 6901 JSON Pointers (/Address/PostCode, /Items/0/Sku) so they round-trip correctly through InputPointer.
